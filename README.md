#NEM-Log 

NEM-Log is a  simple blog, build with Node.js & Express & MongoDB

##Install

Be sure you have installed Node.js/MongoDB/Express

Clone the repo

```
  git clone https://github.com/fiture/NEM-Log.git
```
Install dependencies

```
  npm install
```
